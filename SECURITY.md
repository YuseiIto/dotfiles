# Security Policy
## Supported Versions
This project operates on a "works for me" basis. No formal support is offered.

_Feel free to reach out if you encounter any difficulties! 🌞_

## Reporting a Vulnerability
Vulnerabilities can be reported via email to @yuseiito_ (check the GitHub profile) or, if it's deemed safe to disclose, by posting them on the [issues page](https://github.com/YuseiIto/dotfiles/issues).
